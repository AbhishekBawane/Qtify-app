## Qtify Web

QTify is a modern music browsing web application built with ReactJS that allows users to explore albums, songs, and genres through an interactive and responsive UI. The project uses Material UI, Swiper.js, and REST APIs to create a smooth music discovery experience inspired by modern streaming platforms. 

### Technology
1) React
2) Material UI
3) JavaScript
4) HTML5
5) CSS3
6) Swiper.js

### Features 
1) Browse Top & New Albums
2)  Explore Songs by Genre
3) Search Functionality
4) Fully Responsive Design
5) Interactive Carousels using Swiper.js
6) Fast and Smooth User Experience
7) Modern UI with Material UI
8) Dynamic Data Fetching using REST APIs
9) Reusable React Components

### Tools and Development

3) Git & GitHub
